## Demo Tips

### Before You Start
1. ✅ Have server running: `npm run dev`
2. ✅ Open browser to landing page
3. ✅ Test all features work
4. ✅ Have backup slides ready
5. ✅ Practice transitions between pages
6. ✅ Time yourself (stay under limit)

### During Demo
- **Speak clearly and confidently**
- **Point to specific UI elements** as you explain
- **Use the mouse to highlight** important information
- **Pause after key points** to let them sink in
- **Make eye contact** with audience (not just screen)
- **Show enthusiasm** - you built something cool!

### If Something Breaks
- **Stay calm** - bugs happen
- **Have backup screenshots** ready
- **Explain what should happen** even if it doesn't
- **Move on quickly** - don't dwell on errors
- **Use mock data** - it's already in the code

### Closing Strong
- **Summarize key benefits** in one sentence
- **Call to action** - "Try it yourself at..."
- **Thank the audience** sincerely
- **Be ready for questions** with confident answers
